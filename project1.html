VOIR LE GITHUB



Après avoir terminé l'excellent MOOC "Vue.js" (par Bryan Parasote / Udemy), j'ai eu envie d'utiliser ce superbe framework JavaScript pour créer ma toute première application monopage !

J'ai opté pour un éditeur de police à la fois simple et efficace.

Vue.js est fabuleux pour gérer sans prise de tête les fonctions interactives d'une page web.
Le projet m'a notamment permis de me familiariser avec les directives de classes et de formulaires.

Je me suis particulièrement amusé à créer les boutons de styles, ainsi que la section "couleurs", qui permet de modifier le texte et le background en tapant directement le nom de la couleur désirée !

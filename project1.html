MAQUETTE DE CV





VOIR LE GITHUB



Le premier exercice de ma formation OpenClassrooms consistait à intégrer la maquette d'un CV, avec une mise en page sur deux colonnes.

Pour ce faire, il était possible d'utiliser un framework. J'ai choisi de me servir de Bootstrap 4, principalement pour profiter du système de Grid. Si c'était à refaire, j'utiliserais plutôt la CSS Grid, c'est bien plus simple !

Le projet m'a notamment permis de me familiariser avec les flexbox et l'importation d'icônes Font Awesome.

J'ai géré les couleurs avec des variables CSS (dorénavent j'utiliserais plutôt les variables SASS !).

Enfin, j'ai utilisé pour la première fois des media queries pour rendre un site Responsive.

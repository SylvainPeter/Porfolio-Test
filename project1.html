VOIR LE GITHUB



Pour le premier exercice de ma formation OpenClassrooms, j'ai du intégrer la maquette d'un CV avec une mise en page sur deux colonnes.

Pour ce faire, j'avais la possibilité d'utiliser un framework : j'ai choisi de me servir de Bootstrap 4, principalement pour profiter de son système de Grid. Si c'était à refaire, j'utiliserais plutôt la CSS Grid, c'est bien plus simple !

Le projet m'a notamment permis de me familiariser avec les flexbox et l'importation d'icônes Font Awesome.

J'ai géré toutes les couleurs avec des variables CSS (dorénavent j'utilise plutôt les variables SASS).

Enfin, j'ai utilisé des media queries pour rendre le site Responsive.
